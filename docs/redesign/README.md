# Bridges Institute — Website Repositioning Brief

> North-star doc. Read this first. Every other file in `redesign/` serves the goal stated here.

## The one question the site must answer

**Why would a great English-speaking teacher want to work with Bridges — and how do they apply?**

This is not a redesign. The site already looks good. The change is **purpose**: the homepage now speaks **first to teachers we want to hire**. Schools, partners, parents and donors still matter, but they are a **secondary path lower down the page**.

**The test:** a good English-speaking teacher lands on the site and thinks — *I understand the work, I can picture myself doing it, this looks organized, and I know exactly how to apply.*

## Main message (guides the whole page)

> Bridges is a real, active place to teach English in the Negev. We hire English-speaking educators to help students speak with confidence through movement, games, stories, role-play and real conversation.

## Design guardrails

- Keep the existing palette: **navy, cream, gold** + clean typography. Let real classroom energy carry the page, not decoration.
- **Fewer icons and "perfect" cards.** Icon grids that decorate but don't inform make the site feel AI-built / templated. Say the method **once**, clearly, with real examples and real photos.
- **Real classroom photos** over polished stock. Short caption under each (e.g. "Movement-based lesson, 3rd grade").
- **No fake testimonials.** Only real teacher quotes, or none.
- On mobile, every scroll reveals useful content — **no large blank cream sections**.
- Mobile: **"Apply to Teach" stays easy to reach** (sticky button / sticky bottom bar).

## Words & things to avoid

- Overuse of "connection / transformation / journey."
- Poetic lines that sound nice but don't explain the job.
- Fake or vague testimonials; generic stock-photo feel.
- Icon grids that decorate but don't inform.
- Anything that could belong to a generic English school anywhere — **name the Negev**, real schools, real communities.
